# Strong Inclination Crossword Clue

填字游戏线索查询工具 / Find crossword clues for any word

## 功能

- 输入单词，查找对应的填字游戏线索
- 支持同义词模糊匹配
- 一键复制线索

## 技术栈

- Next.js 14 (App Router)
- Tailwind CSS
- Cloudflare Pages

## 部署

```bash
npm install
npm run build
```

构建产物在 `out/` 目录，可直接部署到 Cloudflare Pages。

## 词库

当前版本内置约 150+ 高频填字词，持续更新中。
