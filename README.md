# Strong Inclination Crossword Clue

填字游戏线索查询工具 / Find crossword clues for any word

## 功能

- 输入单词，查找对应的填字游戏线索
- 支持同义词模糊匹配
- 一键复制线索
- 支持中英文界面

## 技术栈

- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
- Cloudflare Pages (静态部署)

## 部署

```bash
npm install
npm run build
```

构建产物在 `out/` 目录，可直接部署到 Cloudflare Pages。

## 词库

当前版本内置 **16,000+** 填字游戏常用词，包括：
- 基础词汇（DRIVE, LOVE, HAPPY, WORK, LIFE...）
- 派生词汇（各种前缀后缀变形）
- 常见同义词和定义
- 自动生成的线索
